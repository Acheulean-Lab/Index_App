# Electron Web Portal Index

A simple desktop application built with Electron.js to serve as an index for various webpages.

## Getting Started

### Prerequisites

- **Node.js & npm**

### Installation & Running

```bash
git clone https://github.com/Acheulean-Lab/Index_App
cd electrontest

npm install

npm start
```

## Project Structure

- **main.js**: Manages window lifecycle and layout.
- **index.html**: Persistent navigation UI.
- **home.html**: Dashboard grid home page.
- **preload.js**: Secure IPC bridge.
